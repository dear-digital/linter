# linter-ui-proto

## This React app was made using Vite

## Dependencies used:
* Shopify Polaris
* Shopify Polaris Tokens
* i18n

## Steps to install

* In Terminal:
```
npm install 
```
```
npm install @shopify/polaris --save
```
```
npm install i18n --save
```
```
npm run dev
```

More about [Vite](https://vitejs.dev/)
